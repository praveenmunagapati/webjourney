Complete HTML Tag Reference

A comprehensive guide to all HTML5 elements with descriptions, syntax examples, and live previews.

1. Document Structure

<!DOCTYPE html>

Description: Declares the document type and HTML version. Must be the first thing in your HTML document.

Syntax

<!DOCTYPE html>

Notes

This is not an HTML tag, but an instruction to the web browser about what version of HTML the page is written in.

<html>

Description: The root element of an HTML page. All other elements must be descendants of this element.

Syntax

<html lang="en">
  <head>...</head>
  <body>...</body>
</html>

Common Attributes

  • lang - Specifies the language of the document

<head>

Description: Contains meta-information about the document, such as title, scripts, styles, and meta tags.

Syntax

<head>
  <title>Page Title</title>
  <meta charset="UTF-8">
  <link rel="stylesheet" href="style.css">
</head>

Notes

Elements inside <head> are not displayed in the browser window.

<title>

Description: Defines the title of the document, shown in the browser's title bar or tab.

Syntax

<title>My Website</title>

Common Attributes

  • None specific, but accepts global attributes

Preview

Live Preview

Check your browser tab to see the title!

<body>

Description: Contains all the contents of an HTML document, such as text, images, links, tables, etc.

Syntax

<body>
  <h1>Welcome</h1>
  <p>This is the content.</p>
</body>

<meta>

Description: Provides metadata about the HTML document. Metadata is not displayed but used by browsers, search engines, and other web services.

Syntax

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="Free web tutorials">
<meta name="keywords" content="HTML, CSS, JavaScript">

Common Attributes

  • charset - Character encoding for the document
  • name - Name of the metadata
  • content - Value associated with the name
  • http-equiv - HTTP header equivalent

<link>

Description: Defines the relationship between a document and an external resource, most commonly used to link stylesheets.

Syntax

<link rel="stylesheet" href="styles.css">
<link rel="icon" href="favicon.ico">

Common Attributes

  • rel - Relationship between document and linked resource
  • href - Location of the linked resource
  • type - Media type of the linked resource

<script>

Description: Used to embed or reference JavaScript code within an HTML document.

Syntax

<!-- External script -->
<script src="script.js"></script>

<!-- Inline script -->
<script>
  console.log("Hello World");
</script>

Common Attributes

  • src - URL of external script
  • type - MIME type of the script
  • async - Script executed asynchronously
  • defer - Script executed after page parsing

<style>

Description: Used to define CSS styles for a single HTML document.

Syntax

<style>
  body {
    background-color: #f0f0f0;
    font-family: Arial, sans-serif;
  }
  h1 {
    color: blue;
  }
</style>

Common Attributes

  • type - MIME type (usually "text/css")
  • media - What media/device the style is optimized for

<base>

Description: Specifies the base URL/target for all relative URLs in a document. There can only be one <base> element per document.

Syntax

<base href="https://www.example.com/">
<base target="_blank">

Common Attributes

  • href - Base URL for relative URLs
  • target - Default target for links

  <tbody>...</tbody>

<tr>

Description: Defines a row in a table.

Syntax

<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>

<th>

Description: Defines a header cell in a table. Text is bold and centered by default.

Syntax

<table>
  <tr>
    <th colspan="2">Name</th>
    <th>Age</th>
  </tr>
</table>

Common Attributes

  • colspan - Number of columns a cell should span
  • rowspan - Number of rows a cell should span
  • scope - Specifies if header is for column, row, or group

<td>

Syntax

    <td>Data 2</td>
  </tr>
</table>

Common Attributes

  • colspan - Number of columns a cell should span
  • rowspan - Number of rows a cell should span

11. Templates

<template>

Description: Holds HTML content that is not rendered immediately but can be cloned and inserted by JavaScript.

Syntax

<template id="myTemplate">
  <div class="card">
    <h2>{{title}}</h2>
    <p>{{content}}</p>
  </div>
</template>

<script>
  var template = document.getElementById('myTemplate');
  var clone = template.content.cloneNode(true);
  document.body.appendChild(clone);
</script>

Notes

Content inside <template> is not rendered. It's used as a blueprint for creating elements dynamically.

<slot>

Description: Part of Web Components. A placeholder inside a web component that you can fill with your own markup.

Syntax

<template id="element-template">
  <style>
    p { color: blue; }
  </style>
  <h3>My Component</h3>
  <slot>Default content</slot>
</template>

<script>
  customElements.define('my-element', class extends HTMLElement {
    constructor() {
      super();
      let template = document.getElementById('element-template');
      let templateContent = template.content;
      const shadowRoot = this.attachShadow({mode: 'open'});
      shadowRoot.appendChild(templateContent.cloneNode(true));
    }
  });
</script>

<my-element>
  <p>Custom content goes here</p>
</my-element>

Common Attributes

  • name - Slot name for named slots

12. Obsolete Tags

<font>

Description: OBSOLETE - Used to define font face, size, and color. Use CSS instead.

Syntax

<font face="Arial" size="4" color="red">Text</font>

Notes

Deprecated: This tag is obsolete in HTML5. Use CSS for styling: span { font-family: Arial; font-size: 18px; color: red; }

<center>

Description: OBSOLETE - Centered content. Use CSS instead.

Syntax

<center>This text is centered</center>

Notes

Deprecated: Use CSS: div { text-align: center; }

<marquee>

Description: OBSOLETE - Creates scrolling text. Use CSS animations instead.

Syntax

<marquee>This text scrolls!</marquee>

Notes

Deprecated: Never use this. Use CSS animations or JavaScript for moving content.

<big>

Description: OBSOLETE - Makes text bigger. Use CSS instead.

Syntax

<big>Big text</big>

Notes

Deprecated: Use CSS: span { font-size: larger; }

<tt>

Description: OBSOLETE - Teletype text (monospace). Use <code> or CSS instead.

Syntax

<tt>Monospace text</tt>

Notes

Deprecated: Use <code> or CSS: span { font-family: monospace; }
