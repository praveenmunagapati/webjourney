HTML Tags by Category

Complete reference with individual examples for each tag.

Document and Metadata

<html> — Root Element

Wraps all HTML content. Example:

<html lang="en">
  <head>...</head>
  <body>...</body>
</html>

<head> — Metadata Container

Contains title, meta tags, and links. Example:

<head>
  <title>Page Title</title>
  <meta charset="UTF-8">
</head>

<title> — Browser Tab Title

Sets the page title in browser tab. Example:

Look at the browser tab to see the title.

<meta charset> — Character Encoding

Ensures special characters display correctly. Example:

<meta charset="UTF-8">

<meta viewport> — Responsive Design

Makes pages work on mobile devices. Example:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

<meta name> — Page Description

Adds page description for search engines. Example:

<meta name="description" content="My website about web development">

<link> — External Resources

Links stylesheets, icons, and other resources. Example:

<link rel="stylesheet" href="style.css">
<link rel="icon" href="favicon.ico">

<style> — Embedded CSS

Embeds CSS directly in HTML. Example:

<style>
  body { background-color: lightblue; }
  p { color: darkblue; }
</style>

<script> — JavaScript Code

Adds interactivity with JavaScript. Example:

<script>
  console.log('Page loaded!');
  document.getElementById('btn').onclick = function() {
    alert('Button clicked!');
  }
</script>

<base> — Base URL

Sets base URL for all relative links. Example:

<base href="https://example.com/">

<noscript> — No JavaScript Fallback

Shows message if JavaScript is disabled. Example:

<noscript>
  Please enable JavaScript to use this website.
</noscript>

Page Structure

<body> — Main Visible Content

Contains all visible page content. Example:

All visible content goes here.

<header> — Page or Section Header

Typically contains logo, site title, navigation. Example:

My Website

Welcome to the header section

<nav> — Navigation Menu

Contains navigation links. Example:

Home | About | Contact

<main> — Primary Content

Wraps the main content area. Example:

Main Article

This is the primary content of the page.

<section> — Thematic Content Grouping

Groups related content together. Example:

Section Title

This content is grouped in a section.

<article> — Independent Content Block

Represents self-contained content like blog posts. Example:

Blog Post Title

Posted on June 6, 2026

This article stands alone and can be reused elsewhere.

<aside> — Sidebar or Related Content

Typically used for sidebars and related links. Example:

Related Links

  • Related Article 1
  • Related Article 2

<figure> — Self-Contained Content with Caption

Groups images or diagrams with captions. Example:

<figcaption> — Figure Caption

Provides caption for figure. Example:

<figure>
  <img src="photo.jpg" alt="My Photo">
  <figcaption>This is my photo from 2026</figcaption>
</figure>

<footer> — Page or Section Footer

Contains copyright, links, contact info. Example:

© 2026 My Company. All rights reserved.

Privacy Policy

<details> — Collapsible Content

Creates expandable disclosure widget. Example:

Click to expand

This content is hidden until you click the summary.

<summary> — Collapsible Header

Shows header for details element. Example:

<details>
  <summary>More Information</summary>
  <p>Hidden content appears here</p>
</details>

<dialog> — Modal Dialog

Creates dialog boxes. Example:

<dialog open>
  <p>This is a dialog box</p>
  <button>Close</button>
</dialog>

<template> — Content Template

Stores HTML for reuse with JavaScript. Example:

<template id="card">
  <div class="card">
    <h3></h3>
    <p></p>
  </div>
</template>

Text and Content

<h1> to <h6> — Headings

Organize content hierarchy. Examples:

This is H1 - Main Title

This is H2 - Section Title

This is H3 - Subsection Title

This is H4 - Minor Heading

This is H5 - Even Smaller
This is H6 - Smallest Heading

<p> — Paragraph

Wraps text paragraphs. Example:

This is a paragraph of text. It can contain multiple sentences and spans multiple lines. The paragraph tag automatically adds space above and below.

<strong> — Important Text

Emphasizes important words with bold. Example:

This is very important information.

<em> — Emphasized Text

Emphasizes text with italic. Example:

This is emphasized text.

<b> — Bold Text

Makes text bold (non-semantic). Example:

This is bold text for style only.

<i> — Italic Text

Makes text italic (non-semantic). Example:

This is italic text for style only.

<u> — Underlined Text

Underlines text. Example:

This is underlined text.

<s> — Strikethrough Text

Adds strikethrough. Example:

This is old price $50 now $25.

<mark> — Highlighted Text

Highlights text in yellow. Example:

This is highlighted text.

<small> — Small Text

Makes text smaller. Example:

Normal text and smaller text

<code> — Code Snippet

Displays code inline. Example:

Use the function console.log() to debug.

<pre> — Preformatted Text

Preserves spacing and line breaks. Example:

Line 1
    Line 2 with spaces
      Line 3 indented

<blockquote> — Block Quotation

Quotes longer passages. Example:

"The only way to do great work is to love what you do." - Steve Jobs

<q> — Inline Quotation

Short quotes within text. Example:

He said, This is important yesterday.

<abbr> — Abbreviation

Shows full text on hover. Example:

The HTML is used for web pages.

<cite> — Citation

References sources. Example:

Web Development 101 by Jane Doe is a great book.

<address> — Contact Information

Shows contact details. Example:

Contact us:
Email: info@example.com
Phone: (555) 123-4567

<time> — Date/Time

Marks dates and times. Example:

The event is on June 15, 2026.

<kbd> — Keyboard Input

Shows keyboard keys. Example:

Press Ctrl + C to copy.

<samp> — Sample Output

Displays program output. Example:

The command returned: success

<var> — Variable Name

Shows variable names. Example:

Set x = 10 in your code.

<sub> — Subscript

Text below the line. Example:

Water is H2O.

<sup> — Superscript

Text above the line. Example:

Einstein's famous equation: E=mc2

<span> — Inline Container

Groups inline content for styling. Example:

This is red text in a paragraph.

<div> — Block Container

Groups block content. Example:

This content is grouped in a div block.

It can contain multiple elements.

<br> — Line Break

Creates line break. Example:

Line 1
Line 2
Line 3

<hr> — Horizontal Rule

Creates visual separator. Example:

Content above

Content below

<wbr> — Word Break Opportunity

Suggests where text can break. Example:

VeryLongWordThatCanBreakAtThisPoint

<bdi> — Bidirectional Isolation

Handles mixed text directions. Example:

User مرحبا said hello.

<bdo> — Bidirectional Override

Changes text direction. Example:

Normal text reversed text mixed.
